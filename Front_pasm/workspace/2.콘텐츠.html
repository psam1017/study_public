h1, h2 ... h6 : 제목을 나타내는 head 태그. h1의 글씨 크기가 제일 크다.

p(paragraph) : 단락을 구분 짓는 태그
br : 줄바꿈을 나타내는 단독 태그

blockqoute : 인용문을 쓸 때 사용하는 태그. 기본적으로 들여쓰기를 한다.

q : 내용을 인용할 때 사용하는 태그. 들여쓰기 없이 큰 따옴표가 표시된다.
b : 진하게 표시할 구문
string : 진하게 강조할 구문
i : 이탤릭체로 표시할 구문
em : 이탤릭체로 강조할 구문
mark : 형광펜 효과
span : 글자를 꾸밀 때 주로 사용한다. inline 요소를 가지고 있다.
ins : 밑줄을 표현한다.
del : 취소선을 표현한다.
sup : 위 첨자를 나타낸다.1
sub : 아래 첨자를 나타낸다.2




    ol(ordered list) : 순서가 있는 목록
  1. li : list3
  2. list 시작 값이 3
  3. list 종류가 숫자
    ul(unordered list) : 순서가 없는 목록
  • li : list1
  • li : list2
  • li : list3
dl(description list) : 설명이 있는 목록. title과 value 쌍을 사용한 목록이다.
dt : 하나의 제목 태그를 넣는다.
dd : 하나의 제목 아래 여러 개의 설명을 넣을 수 있다.
*하나의 dl 안에는 하나의 dt와 여러 개의 dd를 넣을 수 있다.





caption : 표의 제목
th : 1열 제목 th : 2열 제목
td : 1행 1열 내용 td : 1행 2열 내용
td : 2행 1열 내용 td : 2행 2열 내용
foot에 해당하는 내용
