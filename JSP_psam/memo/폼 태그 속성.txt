폼 태그 속성

action : 폼을 전송할 서버 쪽 스크립트 파일을 지정
target : action에서 지정한 스크립트 파일을 현재 창이 아닌 다른 위치에 열도록 지정

id : css, js에서 사용할 수 있는 태그 선택자
name : 폼을 식별하기 위한 이름(변수). js에서 구분자 '.'으로 접근할 수 있다.
value : 컨트롤 요소의 값

method : 폼을 서버에 전송할 http 메소드를 지정

accept-charset : 폼 전송에 사용할 문자 인코딩 지정
