1. 구조체 타입 선언
(방법1)
struct 구조체 이름{
	자료형 멤버변수명,
	...
};

(방법2) 구조체 별명 사용하기
typedef struct 구조체이름{
	자료형 멤버변수명,
	...
} 구조체별명;	

선언 시 주의사항
-> 함수 밖에서 선언해야 한다.
-> 구조체 선언 시 멤버 변수에 초기값을 넣을 수 없다.



2. 구조체 변수 선언, 생성, 사용
(방법1-1) 선언과 생성을 따로
struct 구조체이름 변수명;
변수명.멤버변수명 = 값;

(방법1-2) 바로 값을 대입
struct 구조체이름 변수명 = {값, ...};

(방법2)구조체 별명을 사용한 경우
구조체별명 변수명 = ...;



3. 구조체 배열
-> 구조체를 배열로 만들 때, 배열의 길이에 변수, 또는 main 안에서 생성된 상수는 들어갈 수 없다. 따라서 main 밖에서 선언한 상수를 사용해야 한다.
-> 구조체의 길이를 임의의 변수로 정하려면 동적 배열 할당이 필요하다.

*구조체 배열 안에 접근
(방법1) 구조체명[i].변수명
-> 구조체의 i번째 자료의 변수에 접근한다.

(방법2) 구조체명->변수명
-> 반복문으로 사용한다면 index를 따로 넣지 않아도 순차적으로 변수에 접근할 수 있다.
-> 반복문 없이 사용한다면 배열 그 자체가 가지는 주소에 접근하는 것이므로 배열의 첫 번째 변수가 출력된다.
	-> *arr == *(arr + 0) == arr[0]
