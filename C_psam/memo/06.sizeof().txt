sizeof()는 매개변수의 메모리 크기를 반환한다.

1. 예를 들어, short는 2바이트, int는 4바이트를 반환한다.

2. 포인터 타입(&var)은 4바이트이다.
-> 따라서 어떤 자료형이든 그 자료형의 주소값은 모두 4바이트를 반환한다.
-> 포인터 변수로 생성한 문자열 또한 문자열 길이에 상관 없이 포인터 타입 변수의 크기를 알려주기 때문에 4바이트를 반환한다.
-> 배열로 생성한 문자열은 널 문자를 포함하여 문자열의 길이를 반환한다. 하지만 문자열의 주소값은 여전히 4바이트를 반환한다.

3. scanf_s("서식문자", 변수의 주소, sizeof(변수))

// iVar은 정수형 변수
-> scanf_s("%d", &iVar, sizeof(iVar))
	-> sizeof 함수의 매개변수로 iVar의 크기를 전달해야 한다. iVar의 주소 크기는 자료형의 크기 상관 없이 4바이트이기 때문에 적절하지 않다.

// str은 배열로 생성한 문자열
-> scanf_s("%s", str, sizeof(str));
	-> 배열로 생성한 문자열 자체가 주소를 의미하므로 str에 바로 값을 입력하면 된다.
	-> sizeof 함수에는 위와 동일하게 주소의 크기가 아니라 자료형의 크기를 전달해야 한다.
